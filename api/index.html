Endpoint Definitions

Add Employee (POST)

What: This Employee Navigator push resource allows a payroll partner to receive a full individual employee demographic record.

When: It is used when a new hire is processed in Employee Navigator.

Update Employee (POST/PUT)

What: This Employee Navigator push resource allows a payroll partner to receive a full individual employee demographic record upon an employee update.

When: For example, if an established employee has an address update in Employee Navigator, this resource will be used.

Payroll Deduction (POST)

What: This Employee Navigator push resource allows a payroll partner to receive benefit deduction information from Employee Navigator when data changes in the Employee Navigator system.

When: For example, if an employee has a life event that results in a new deduction amount, this resource will be used.

Get Employee (GET)

What: This payroll partner push notification allows Employee Navigator to receive notifications from our payroll partners when a new hire is created or one of the agreed upon data exchange fields of an individual employee demographic record has changed in payroll.

When: For example, if an employee has an update to their middle name in payroll, this resource will be used.
